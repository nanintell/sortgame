Bubble Sort


The algorithm uses Decrease and Conquer Strategy and its Big-O is O(n2) for average and worse case but in best case, it becomes O(n).
Best Case of Bubble Sort in ascending order is ascendingly sorted data while Worst Case is descendingly sorted data
Sorting in ascending order (lowesr on left and highest on right), using Bubble Sort is done by comparing 2 values next to each other.
If left value is higher than right value, swap them. Once all pairs are compared and/or swapped, the rightmost value is the highest one.
What if we repeat this method once more but stop at the value before the rightmost?
...Right! The value before the rightmost is the second highest value!
Thus, if we keep repeating and decreasing stopping point, we get the sorted data.

First Loop As seen in the pictures, the highest value which is 5 is being repeatedly swapped to rightmost position.

Second Loop The time, we only compare to the value before rightmost.

Last Loop On the last loop, we compare only 1 pair. Once done, we get the sorted data.
