Distribution Counting Sort


This algorithm's Big-O is O(n + k) for all cases where k is maximum value. Best Case for sorting in ascending order is sorted data in ascending order while Worst Case is sorted data in descending order.
Distribution Counting Sort is done by finding frequency of each value then calculating cumulative frequency. These frequency can be used to tell the correct position of sorted data. Thus, this algorithm is good for data with a lot of duplicate values.
Once done, loop bringing each value into a position matched with cumulative frequency of that value. Then, decrement the frequency by 1 and move on.
