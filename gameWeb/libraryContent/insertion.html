Insertion Sort


The algorithm uses Decrease and Conquer Strategy and its Big-O is O(n2) for average and worse case but in best case, it becomes O(n). Just like Bubble Sort.
Best Case of Insertion Sort in ascending order is ascending sorted data while Worst Case is descending sorted data.
Insertion Sort (ascending) is like separating data into sorted data and unsorted data
First, we let the sorted data is the leftmost data while the rest is unsorted data. After that, We compare the value next to sorted data. If that value is less, meaning that it's not sorted, we swap to left.
Now, we have 2 values in sorted data! Then, let's continue. If it's less, we swap.
But! If that value is still less than the value of the left?
We simply compare the value with the left one then if value is less than the left, swap them. Repeat this until the left value is less than right one.
Then, we keep sorting in unsorted data until no unsorted data left.
Finally, we get ascending sorted data. (left is lowest and right is highest)

Once done, we get the ascending sorted data
